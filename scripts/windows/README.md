# Windows overlay helper

## Quote Chaser metrics overlay

File:
- `quotechaser-metrics-overlay.ahk`

## What it does

Creates a small always-on-top desktop overlay that polls:
- `https://quotechaser.online/api/project-metrics`

It shows:
- live users
- signups today

## Setup

1. Install AutoHotkey v2 on your Windows PC.
2. Copy `quotechaser-metrics-overlay.ahk` to your PC.
3. Double-click the script.

## Hotkeys

- `Ctrl+Alt+R` refresh now
- `Ctrl+Alt+O` move back to the top-right corner
- `Ctrl+Alt+Q` close the overlay
